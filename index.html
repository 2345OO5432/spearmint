Spearmint

An updated id Tech 3 engine for continuing the classics and creating new games.

View project onGitHub

Spearmint is a ioquake3 based game engine that has two main goals;

  1. provide a flexible engine for creating new games and mods.
  2. support features from (and running) various id Tech 3-based games.

Some of the major features are:

  • Four player splitscreen
  • Moved a lot of code from server and client to Game and CGame VMs;
    • including console, chat input overlay, usercmd_t creation, and a lot of bot AI code.
  • Allow games to modify entityState_t and playerState_t without changing the engine.
  • Extended API for Game and CGame/UI, includes some of the features from;
    • RTCW, WolfET, Tremulous, Turtle Arena, and World of Padman
  • Merged UI VM into CGame VM, easier to modify as a whole / more flexible.
  • Additional shader keywords.

Read more at the Spearmint wiki.

Running Quake III Arena using Spearmint

  • Get the source and compile it using make. The game will be in build/release-YOURPLATFORM-YOURARCH/.
  • Copy your Quake III Arena pak[0-8].pk3 into build/release-YOURPLATFORM-YOURARCH/baseq3. (Don't own Q3? Buy it on Steam.)
  • Get the Spearmint Q3A patch files from here and put in build/release-YOURPLATFORM-YOURARCH/baseq3/
  • Run build/release-YOURPLATFORM-YOURARCH/spearmint_YOURARCH

You can add splitscreen players after you start or join a server using the in-game menu.

Running Quake III: Team Arena using Spearmint

  • Follow the steps for Quake III Arena above.
  • Copy your Quake III: Team Arena pak[0-3].pk3 into build/release-YOURPLATFORM-YOURARCH/missionpack. (Don't own Team Arena? Buy it on Steam.)
  • Get the Spearmint Q3TA patch files from here and put in build/release-YOURPLATFORM-YOURARCH/missionpack/
  • Run build/release-YOURPLATFORM-YOURARCH/spearmint_YOURARCH and choose Team Arena from the menu.

The UI currently lacks support for splitscreen. You can add splitscreen players after you start or join a server using the console (open/close using shift+esc or ~). 2dropin, 3dropin, and 4dropin commands add splitscreen players. 2dropout, 3dropout, and 4dropout commands remove splitscreen players. You'll need to manually bind controls, example bind u +2forward to have u move player 2 forward. Type +2 then press tab to see full list.
